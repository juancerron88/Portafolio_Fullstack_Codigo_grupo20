Mi Proyecto Final

El Proyecto es "Gestión de Activos con Tecnologías 4.0", pretende alcanzar a diseñar un entorno para 03 niveles de usuario: Operario de Mantenimiento, Jefetura de Mantenimiento y Gerencia de Producción, con el propósito de desarrollar un tablero de control donde permita al usuario analizar información proveniente de las máquinas industriales (Temperatura, vibración, alineamiento y consumo energético) donde se pueda garantizar la disponibilidad de los equipos, implementar estrategias de mantenimiento preventivo/predictivo y analizar indicadores de producción.

En esta primera etapa del proyecto (FRONTEND) mostraremos dashboard para la jefatura de mantenimiento donde se mostrarán datos estáticos de Disponibilidad de Equipos, Presupuesto de Mantenimiento, Costos de Mantenimiento y Lista de Equipos Críticos.

Usuarios Finales

Operarios de Mantenimiento

Actividades relacionados a monitoreo a condición de los equipos, donde podrá analizar vibración, temperatura, alineamiento, etc, desde el panel de control.

Jefatura de Mantenimiento

Puede agregar equipos y aplicar e implementar RCM (Mantenimiento Basado en la Confiabilidad) ingresando niveles de criticidad por cada equipos, así mismo programar actividades de mantenimiento preventivo.

Gerencia de Operaciones

Analizar indicadores estratégicos de producción, como mermas, tiempos muertos, consumo energético, costos de cada producto según el impacto del mantenimiento.

Flujo de Trabajo del Desarrollo Web

En el proyecto se ha realizado el siguiente flujo de trabajo.

  • Diseño de la Maqueta en FIGMA
  • Instalación de Librerías (Dependencias)
  • Creación de Mockdata
  • Desarrollo de Componentes para el entorno del Usuario
  • Integrar API Key (token) para UBIDOTS
  • Llevar a producción del Proyecto en Github

Mejoras del Proyecto Frontend

En el Proyecto Frontend considero que hay mejoras por implementar en la experiencia del usuario final como:

  • Agregar niveles de más Usuarios considerando la UI/UX
  • Agregar rutas seguras y privadas por niveles de usuario
  • Ingresar Layout interactivo de los procesos industriales
  • Habilitar periféricos como cámaras para poder realizar inspecciones via streaming
  • Desarrollar APIs de los sensores que tendrá cada equipo
  • Desarrollar la descarga de reportes por niveles de usuarios
