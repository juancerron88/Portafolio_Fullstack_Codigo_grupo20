Bienvenido a mi Website¡¡¡

En este curso de especialización aprenderé a reforzar mis competencias en Desarrollo Web para ello este repositorio será para mostrar mis apuntes en cada uno de los temas que se abordarán. Los docentes de este primer módulo son: Guillermo Sifuentes, Jorge Garnica y Mariana Navarro. El curso FullStack se compone de 02 módulos Frontend y Backend.

Frontend

El desarrollo web front-end consiste en la conversión de datos en una interfaz gráfica para que el usuario pueda ver e interactuar con la información de forma digital usando HTML, CSS y JavaScript

Temario Frontend:

  • Github, HTML5 y CSS
  • Fundamentos de CSS
  • Responsive Website Design y Flexbox
  • Fundamentos de JavaScript
  • Objetos Métodos Manipulando en DOM
  • Eventos y Localstorage
  • Bootstrap y Librerías en JavaScript
  • ECMAscript6
  • Fundamentos de React
  • Proyecto React

Backend

El backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Se trata del conjunto de acciones que pasan en una web pero que no vemos como, por ejemplo, la comunicación con el servidor.

Temario Backend:

  • Python sintaxis Semantica
  • Desarrollo Web Manejo de APIs
  • Base de Datos Relacional
  • Web Django
  • Servicios en Django
  • Pasarela de Pagos, MongoDB y Firebase
  • Cloud Computing Services
  • Nodejs
  • Microservicios
  • Introducción a Bigdata & MLPython
